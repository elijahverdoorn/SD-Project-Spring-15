The Character Class Specifications

This class is used for creating a character, which is put on the game map and can interact with objects and tiles.

Elijah Verdoorn; XML created by Elijah Verdoorn on May 15, 2015




CLASS Tile

No superclass.

State variables:

private Sprite* sprite
This is used to get the sprite used for the tile.

private Texture texture
This is used to set the texture used for the sprite of the tile.

private int posX
We use this to get what X position on the grid the character is at.

private int posY
We use this to get what Y position on the grid the character is at.

private int gridLocation
This will be used for calculating the array position of the character

private bool selected
This is used for knowing if the specific character is selected by the user.

private std::string name
We use this for giving the character a distinct name.

private int maxSpeed
Tracks the characters maximum speed.

private int currentSpeed
Holds the characters remaining speed as they move.

private int teamID
Holds the character's team number.

private int maxHitPoints
Holds the characters maximum hit points.

private int maxAtkPower
Holds the characters maximum attack power.

private int maxDefPower
Holds the characters maximum defense power.

private int maxRange
Holds the characters's maximum attack range.

private int maxActions
Holds the characters maximum actions.

private int currentHitPoints
Holds the character's remaining hit points.

private int currentAtkPower
Holds the character's remaining attack power.

private int currentDefPower
Holds the character's remaining defense power.

private int currentRange
Holds the character's remaining range.

private string fileName
Holds a filename.

private bool alive
True when hit points are above 0.

void move
Moves a character

Arguments:

int par
How much a character moves

int dir
Which way a character moves; Up = 1, Right = 2, Left = 3, Down = 4

State change:
Moves a character par tiles in dir direction

No return value.
