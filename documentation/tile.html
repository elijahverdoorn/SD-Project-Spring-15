CLASS Tile

No superclass.

State variables:

*sprite sprite
A sprite object

Texture texture
The associated Texture Object

int posX
The X position of this tile

int posY
The Y position of this tile

int width
The width of this tile, in pixels

int height
The height if this tile, in pixels

string fileName
The filename associated with this tile

Constructors:

public Tile
The default constructor.

No arguments.

State change:
This instance of Tile is initialized with no texture, sprite or other values.

public Tile
Constructor from a Texture

Arguments:

Texture par
The texture to be used

State change:
The texture is assinged to Texture, and a sprite is created from the Texture.

public Tile
Constructor from a filename

Arguments:

string fileName
The file to pull the texture from

State change:
The texture is created from the fileName, then the sprite created from the texture.

Destructor:
~Tile
No Destructor Required
